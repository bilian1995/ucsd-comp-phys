To compile/recompile binaries used by the scripts, do:

make

OR

make appName

where appName is one of the following:
aarseth

Note:
'make clean' will delete the binaries (to start from scratch)

_____
To run script, open MATLAB (or GNU Octave) and navigate to the 'MLAB' folder in this directory,
do (in the MATLAB/Octave command line):

projectScript

OR

Open the projectScrip.m file located in the 'MLAB' folder and press the 'Play' button to run the script (don't forget to navigate to the 'MLAB' directory first.
