To compile/recompile binaries used by the scripts, do:

make

OR

make <app name>


To run script, open MATLAB (or GNU Octave), navigate to the 'MLAB' folder in this directory
and do (in the MATLAB command line):

projectScript

OR

Open the projectScrip.m file located in the 'MLAB' folder and press the 'Play' button to run the script.
